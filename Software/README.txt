Place contents of this folder in the root directory where you want to search, i.e. U:\eng_research_roblyer\ or any folders within.

Start from executable
1) Run app.py

Start from Command Prompt or PowerShell
1) Open Command Prompt or PowerShell
2) Navigate to the inside of this directory
3) Start server with the command "python -m CGIHTTPServer 80"

Once started, access webpage from http://[server IP address]

Note: once started, the webpage can be accessed locally on the server from http://localhost
